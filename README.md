## Evaluation Criteria

### Styling

- **B1** A global reset for `<h1>...<h6>`, `<p>`, and `<ul>` is allowed.
- **B2** No margin collapsing outside parent elements.
- **B3** Vertical spacing between elements is set using `margin`.
- **B4** Space between parent border and child is set using `padding`.
- **B5** Margin and padding values match the layout exactly.
- **B6** A utility class `.container` is created for centering and limiting width.
- **B8** Container width is **1158px**.
- **B9** The container wraps header, footer, and sections content.
- **B10** Flexbox is used only where necessary (header, navigation, lists, etc.).
- **B11** Final element sizes match the layout.
- **B12** No fixed heights; height depends on content.
- **B13** Header has a subtle bottom border.
- **B14** Sections are stacked vertically without margins.
- **B15** All sections use `.section` with `padding: 120px 0;`.
- **B16** Portfolio cards have a bottom border only.
